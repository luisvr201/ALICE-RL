Help

Creando un nuevo mapa y configurando el nombre del mapa:
  • Cambie la URL a archivo:///maidmap.html?map=[map_name]
  • Este formato fue elegido para agregarlo fácilmente a favoritos en su navegador web.
Interactuando con el mapa mental:
  • Haga clic con el mouse para seleccionar un nodo, haga doble clic para editar, presione Enter para guardar
  • Puede hacer clic y arrastrar el mouse para desplazarse por el mapa mental, o usar la rueda del mouse o los movimientos con dos dedos en el panel táctil para desplazar el mapa mental vertical u horizontalmente si es más grande en esas dimensiones (respectivamente) de la pantalla.
  • Arrastre los nodos con el mouse para pasar a otro nodo
  • Haga clic con el mouse en un círculo pequeño para expandir/contraer
  • Presione Enter para crear un sub nodo
  • Presione Ctrl-Enter para crear un nodo secundario.
  • Presione Suprimir para eliminar un nodo
Tenga en cuenta que los mapas se guardan automáticamente a través de la API IndexedDB, un almacenamiento basado en navegador al que solo puede acceder el mismo origen. En otras palabras, solo se puede acceder a sus mapas desde donde se proporciona esta página. Tenga cuidado al utilizar otras aplicaciones basadas en navegador que guarde o sirva desde la misma ubicación, ya que pueden acceder al almacenamiento de la API de IndexDB. Utilice únicamente código que sea abiertamente revisable, de código abierto o de una entidad confiable.

Barra de botones:
  • - Make a backup of the map. As mentioned above, the map is already stored in your browser, but this can be used for transferring between computers.
  • - Load a backup of the map. The map must be empty to ensure you don't accidentally overwrite another map.
  • - Get suggestions from GPT-3.5, which is what this is all about! You can configure how this is done in settings.
  • - Collapse all
  • - Expand all
  • - Zoom in
  • - Zoom out
  • - Engage in a deeper discussion with GPT-3.5 about a particular node. The first time you chat about a node both the mindmap and a series of questions will be sent to GPT API. Subsequently, all messages will be prefixed with the mindmap only. You can configure these prompts in settings.
  • - Undo, may have to click twice. This is a bit hard to use right now, working on it.
  • - Redo, may have to click twice
  • - Settings. You can change the prompts used to get suggestions and discuss nodes. Make sure you review them carefully before making changes. Backing up a copy of the prompt for reverting is probably a good idea.
